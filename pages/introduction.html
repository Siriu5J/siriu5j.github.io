Course Introduction

Course Description

As you might have known, A.C.E. does provide a computer science project that is worth 0.5 high school credit. However, I have noticed that the project itself lacked tutorial and detailed guidance on how to utilize the powerful computer productivity software suite. The project is focused on the application side of using computer technology. This alternate computer science elective is based on the A.C.E. computer science project and is meant to focus heavily on the tutorial side. Students are expected to learn the text, take quizzes, and then complete the A.C.E. computer science project with extra requirements that aims to utilize the skills learned from the tutorials.

In this course, Microsoft Office would be used as the main software suite. However, since most productivity suites resembles one another in many ways, it is possible to complete this course without using Microsoft Office, but please be reminded that the student might need to conduct extra research on certain features that might be offered in a different way on another software suite.

This computer science elective will be divided into two main topics: computer productivity suite and internet applications. The GCF Learn Free office tutorial will be used as the text for the productivity suite part of this elective while various other online resources will be used for the internet applications part.

My Goals

  • To learn the basics of computer word processing
  • To learn to edit reports and essays with word processors
  • To learn to cite and create bibliographies with word processors
  • To learn the basics of computer slideshow creation
  • To learn to use slideshow as a visual aid for a speech
  • To learn to utilize animations in a slideshow
  • To learn the basics of computer spreadsheet
  • To learn to add, edit, and process data on a spreadsheet
  • To learn the basics of spreadsheet functions
  • To learn to conduct efficient research online
  • To learn to determine the quality of your research data
  • To learn to utilize email services

How to complete this course

To complete the course, the student is required to go through the GCF Learn Free tutorials, complete the practice problems and quizzes, and complete the projects. The practice problems are ungraded and are aimed to help the student comprehend and consolidate the materials learned. Therefore, most questions would be open-ended, and sample answers would be provided. The quizzes are available on GCF Learn Free; record the scores from the quizzes. Some of the simpler projects would be single-stepped project with simple goals while other ones might contain multiple parts and would be way more time-consuming. A progress card can be downloaded at the resource page of this website. To complete the course, a student would want to aim for around 1 hours per week for a semester or 2 hours per week for a quarter.
