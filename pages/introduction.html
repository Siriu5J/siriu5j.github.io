Course Introduction

You can find a list of software tools that you will need to complete this project.

You are not required to select software tools from this following list. This list is only here to provide some suggestions that you might be interested in. In this course, the tutorials will be using Microsoft Office and Google Chrome. Please note that some of the software in this list are non-free software.

Office Suite

  • Microsoft Office Software Suite
  • Microsoft Office Online
  • WPS Office
  • LibreOffice
  • Apple iWork Productivity

Web Browser

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge (Windows 10 built in)
  • Apple Safari (MacOS built in)

Email Program

  • Mozilla Thunderbird
  • Microsoft Mail (Windows 10 built in)
  • Apple Mail (MacOS built in)
  • Microsoft Outlook
  • Other Web-based Email Providers
